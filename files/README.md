# Diagnóstico Cloe.com — React + Vite

Reporte ejecutivo del canal online Cloe.com construido con React + Vite.

## 🚀 Deploy en Vercel

### Desde GitHub (recomendado)
1. Sube esta carpeta a un repo en GitHub
2. Entra a [vercel.com](https://vercel.com) → **Add New Project**
3. Importa el repo — Vercel detecta Vite automáticamente
4. Click en **Deploy**

### Vercel CLI
```bash
npm install
npm run build
npx vercel --prod
```

## 🛠 Desarrollo local
```bash
npm install
npm run dev
```
